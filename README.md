# Шифратор (Encryptor)

Простое десктопное приложение для шифрования и дешифрования текста. Программа разработана на C++ с использованием фреймворка Qt 6.

## Возможности

- **Графический интерфейс:** простое окно с двумя текстовыми полями (ввод и вывод) и темной темой.  
- **Работа с файлами:** поддержка загрузки исходного текста из `.txt` файла и сохранения результата в `.txt` файл.  
- **Выбор метода:** позволяет пользователю выбрать один из нескольких методов шифрования.

## Реализованные методы

Программа включает 5 различных методов шифрования:
1. Сдвиг нечетных символов влево  
2. Переворот слова + сдвиг  
3. Замена первых 2 букв (таблица) — использует `table.txt`  
4. Четный символ + код из файла — использует `method4_key.txt`  
5. Блоки (5) — перестановка 1,2 и 4,5  

## Сборка

Проект использует CMake для сборки.

## Требования

- Компилятор C++17  
- CMake версии 3.21+  
- Qt 6 (Core, Widgets)  

## Шаги для сборки

```
1. Клонируйте репозиторий
git clone [URL_ВАШЕГО_РЕПОЗИТОРИЯ]
cd Encryptor

2. Создайте директорию для сборки
mkdir build
cd build

3. Сконфигурируйте проект
cmake ..

4. Соберите проект
cmake --build .
```
