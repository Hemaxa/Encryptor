# Шифратор

<img width="1012" height="740" alt="Снимок экрана 2025-11-07 в 16 20 26" src="https://github.com/user-attachments/assets/6b8d8a90-0dea-4c75-8115-646c4a5144d0" />

Простое десктопное приложение для шифрования и дешифрования текста. Программа разработана на C++ с использованием фреймворка Qt 6.

## Возможности

- **Графический интерфейс:** вертикальный интерфейс с панелью управления и текстовыми полями, оформленный в салатовой теме.
- **Работа с файлами:** поддержка загрузки исходного текста из `.txt` файла и сохранения результата в `.txt` файл.
- **Выбор метода:** позволяет пользователю выбрать один из нескольких методов шифрования.

## Реализованные методы

**Программа включает 5 различных методов шифрования:**
1. Код символа - 1
2. Перестановка нечетных символов
3. Замена последних 3 букв (таблица) — использует `table.txt`
4. Нечетный символ (разница) + ключ — использует `method4_key.txt`
5. Переворот блоков (3) 

## Сборка

Проект использует CMake для сборки.

## Требования

- Компилятор C++17  
- CMake версии 3.21+  
- Qt 6 (Core, Widgets)  

## Шаги для сборки

```
1. Клонируйте репозиторий
git clone https://github.com/Hemaxa/Encryptor.git
cd Encryptor

2. Создайте директорию для сборки
mkdir build
cd build

3. Сконфигурируйте проект
cmake ..

4. Соберите проект
cmake --build .
```
